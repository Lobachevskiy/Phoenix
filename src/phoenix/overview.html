An unofficial Java clone/remake/port/patch of/for the game Emperor of the Fading Suns (EFS).

Introduction

An unofficial Java clone/remake/port/patch of/for the game Emperor of the Fading Suns (EFS). Uses original EFS data files and requires EFS 1.4 to be installed. Should work with all mods. All public classes, that is the main classes of the source files, should have at least one sentence of documentation in this API documentation (note that this is not an API per se, but a documentation of the sources, as Phoenix is supposed to be a full application, minus the data files.)

Program structure

The source code is organized as follows:
  • Main program entry point: class Phoenix in package phoenix
  • Graphical user interface implementation: package gui
  • State machine for controlling GUI: package state
  • Most of game data structures and logic: package game
  • Reading in and representing data in GALAXY.GAL-files: package galaxyreader
  • Reading in and representing data in text mode data files: package dat
  • Utility classes: package util
  • Most of AI related data structures and logic (very much WIP): package ai
